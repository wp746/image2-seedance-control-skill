# Image2 Seedance Control Prompts — 多段长剧情测试

## Project Settings
- 片名: 回家 / Coming Home
- 类型: 心理惊悚 / 慢燃恐惧
- 画幅: 16:9
- 总时长: 60 秒 (4 段 × 15 秒)
- 风格: 自然主义恐怖 — 不靠跳吓，靠累积的"不对"。罗曼·波兰斯基式公寓幽闭。Gordon Willis 式阴影。无声的空间比有声的更可怕。
- 生产策略: 1 组资产板 + 项目连续性圣经 + 4 段故事板（各配 Seedance prompt ≤2000 字符）+ 段间接缝检查

## Usage
按 v1.2.1 流程：先出资产板锁定角色和空间，再按 SEG_01→SEG_04 逐段出故事板和 Seedance prompt。Seedance 生成时，每一段都要把上一段的末帧作为参考上传，形成跨段连续性。

## Production Control / 工业化连续性控制

- 类型: 心理惊悚
- 时长: 60 秒

### 角色连续性矩阵

CHAR_A / Lin
- 年龄感: 34-37 岁，亚洲女性。眼角有生活痕迹——不是皱纹，是疲惫的细线
- 脸部锚点: 鹅蛋脸偏窄，颧骨略高但柔和。单眼皮，瞳色深棕近黑。鼻梁直，鼻头微圆。嘴唇偏薄，下唇比上唇略厚

### 情绪递进弧线
SEG_01: 疲惫 → 日常 → 微顿（灯怎么亮着？）→ 回头确认

SEG_03: 走向卧室 → 门口停住 → 看镜子 → 不看自己，看身后 → 身后什么都没有，但恐惧已在
SEG_04: 转身确认无人 → 衣柜门开了 → 后退 → 瞳孔放大 → 静止。没有答案。恐惧不需要答案。

### 声音/VFX 圣经

### Prompt A01 — Character Asset Board — CHAR_A / Lin

```text
这是给 Seedance 2.0 读取的角色资产连续性板，用于 60 秒心理惊悚短片「回家」。不是通用模型表，不是均匀网格，不是海报。

CHAR_A / Lin
身份：34-37 岁亚洲女性，独居，公司职员。有一张被日常生活磨损的脸——不是不美，是美被疲惫覆盖了一层。
年龄感：眼角有细线——不是鱼尾纹，是长期盯屏幕和睡眠不足留下的疲惫痕迹。额头有极淡的抬头纹（只在扬眉时可见）。法令纹浅但存在。嘴唇偏干，有轻微唇纹。
身高体型：165cm，偏瘦——肩胛骨在衬衫下隐约可见。但肩部有力，不是柔弱型。体态微驼（桌前工作习惯），紧张时反而脊背挺直——像动物警觉。

版式要求：非对称艺术指导排版——左侧转身图，中部头部研究和微表情序列，右侧电影肖像和动态姿态，底部材质/比例/色卡/心理档案/禁止漂移。所有标签大号可读。

转身图：全身正面（正面站立，双手自然垂，低马尾，衬衫领口第一颗扣未扣）→ 3/4 前侧（微侧身，可见肩胛骨轮廓）→ 纯侧（鼻子和下颌线条，单眼皮侧面轮廓）→ 背面（马尾、大衣肩线、裤线）→ 动态角度（回头——模拟 SEG_01 中回头确认厨房灯光的那个瞬间）。所有视图同一人物同一服装。

面部设计：真实亚洲肤质——颧骨和鼻翼两侧毛孔可见，T 区轻微油光（下班后未补妆）。颧骨有微血管泛红（不是腮红，是皮肤自己的色调）。下眼睑有轻微暗沉（睡眠不足）。嘴唇偏干，下唇有一道细微竖纹。不是 AI 塑料皮。
眼部关键——单眼皮，眼裂偏窄，瞳色深棕近黑。睫毛直而不翘。内眼角有轻微红血丝。

头部研究：中性正面 → 3/4 侧（这个角度最能体现她的面部结构）→ 纯侧 → 微低头（看钥匙碗）→ 微抬头（看向厨房灯光）→ 转头回望（SEG_01 关键姿态）

微表情范围：
- 基准面：疲惫但平静。眼睑比完全放松略垂 1mm（累）。嘴角自然，不笑不垮。眼神有焦距，不散。
- 第一丝不安：眉毛内端微升 1mm（不是惊恐，是不信）。下唇收紧 0.5mm。瞳孔不变——还没有威胁需要放大瞳孔。
- 确认异常：眼睑抬起 1.5mm（完全警觉）。嘴唇微启 1mm——想说"这不对"但没有声音。呼吸暂停 1.2s（胸腔不动的瞬间）。瞳孔开始缓慢放大。
- 恐惧：眉弓整体下降 2mm（保护反应）。眼睛比基准面睁大 2mm——露出更多眼白。嘴唇紧闭——不抖，是控制。下颚肌肉微绷（咬肌隆起 1mm）。瞳孔扩大至基准面的 1.3 倍。
- ⚠️ 关键约束：从不安到恐惧的递进必须像温度计缓慢上升，不能跳变。每个微表情之间有 0.3-0.5s 的过渡时间。

心理档案：核心特质——理性、善于观察、习惯独处。内在冲突——"我的理性告诉我没什么，但我的身体已经在害怕了"。行为模式——先看、再确认、然后才反应。不是一惊一乍的人，所以当她真的害怕时，那个恐惧是真实的。

表演指导：不是"表演恐惧"。是"压制恐惧然后失败"。Lin 面对异常的第一反应是拿理性去套——灯亮着？可能早上忘了关。窗户开着？可能早上通风忘了。水杯？可能记错了。但当证据累积到一个临界点，理性坍塌——这个坍塌必须在脸上、在呼吸里、在瞳孔的缓慢放大里可见。但幅度极小——这是给镜头 1 米内的特写拍的，不是给舞台最后一排拍的。

服装锁定：
- 深灰羊毛大衣：过膝长度，单排扣（此刻未扣），翻领，两侧口袋。肩部有室外冷空气残留（刚到家的状态）。羊毛质感——细看可见纤维纹理，不是光滑化纤。
- 内搭白色棉质衬衫：领口第一颗扣子未扣（下班放松状态），袖口扣子扣着。棉质有轻微皱褶（坐了一天办公室）。腋下和背部有极轻微汗渍痕迹（不是湿，是穿过一天的自然状态）。
- 黑色直筒西裤：羊毛混纺，膝盖处有坐痕。裤线可见但不锋利。
- 黑色平底皮鞋：皮质，鞋面有穿着折痕。鞋底与木地板接触有皮革摩擦声。
- 银色细链手表：左手腕，表盘约 2cm，简约款。表带贴合但不过紧。
- 黑色皮质通勤包：右肩背，包带在右肩留下微压痕。

电影肖像：在公寓走廊中——身后是刚关上的门（门铰链缺油的低吟尚在空气中），前方厨房吊灯暖黄光照亮她面部的 3/4。她的眼神——看向厨房灯光的方向，有疑问但没有恐惧（还没到恐惧）。50mm f/1.4，焦点在右眼。Kodak 500T 色彩——阴影偏青绿，肤色暖中带冷。颗粒在阴影中可见。

颜色板：深灰（大衣）→ 白（衬衫）→ 黑（裤子、鞋、包）→ 暖黄（厨房灯光）→ 暖肤色（Lin 的皮肤）→ 银（手表）
```

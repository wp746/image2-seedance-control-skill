# Film Industry Master SOP — 影视工业级母版标准

这份文档定义了 image2-seedance-control-skill 的**终极质量标准**。它不是可选参考——它是每一段 Seedance prompt 和每一块 Image2 故事板必须对照的母版检查清单。

如果你发现这个清单里有任何一项在当前 skill 中没有被覆盖，那就是缺口。补上。

---

## 一、制片统筹 / Producing

### 1.1 模型能力边界
Seedance 2.0 的真实能力范围决定了我们能要求它做什么、不能要求它做什么。不了解边界的设计是空中楼阁。

**已知上限:**
- 单段最长 15 秒，超过需分段
- 文字提示词上限 2000 字符
- 图像参考支持多图上传（资产板 + 故事板 + 上段末帧）
- 人物面部在 1m 距离内可保持身份一致（超过 3m 开始漂移风险增加）
- 手持运镜模拟自然但有随机性（需要精确描述抖动特征）

**已知下限:**
- 远距离人群不可控（会随机生成面部）
- 快速复杂动作（翻跟头、高速旋转）漂移风险极高
- 多人物同框超过 2 人时身份漂移概率指数上升
- 复杂 UI 界面文字不可靠（会乱码）
- 精确字幕/标题不可靠

**设计原则:**
- 把 Seedance 当"一个非常有天赋但经验不足的摄影师+演员"，需要极其精确的指导
- 在一个镜头内只让它做一件事——一个主动作、一个情绪变化、一个运镜动机
- 长段落分段不是为了凑 15 秒，是因为 15 秒后身份漂移风险增大

### 1.2 项目管理
- 项目圣经在第一个提示词之前完成
- 资产编号贯穿始终不变化
- 段间手递状态在上一段生成后可能需要根据实际输出微调

## 二、美术指导 / Production Design (Art Department)

美术指导不只是"房间长什么样"。美术指导回答"这个空间在说什么"。

### 2.1 色彩体系
每一部影片有它的色彩宪法。

**需要锁定:**
- 主色调（1-2 个主导色，不是一堆颜色）
- 辅色（1-2 个对比或互补色）
- 强调色（只出现在关键道具或关键时刻的少量高饱和色）
- 中性色（灰/白/黑/米——空间基底）
- 禁止色（某些颜色在这个世界里不存在）

**色彩叙事:**
- 暖色前推（亲近、威胁、热情）、冷色后缩（疏离、冷静、恐惧）
- 同一场景的色彩随情绪变化——不是整体变暖或变冷，是特定颜色的进退
- 道具颜色是叙事——一个红色物体在灰色世界里是信号

**实现方式:**
- 资产板的色卡区域必须标注主色/辅色/强调色/中性色/禁止色
- Seedance prompt 中指定"本段色彩宪法"——不是"暖色调"，是"2700K 暖黄主动光 + 2000K 钠灯橙补光 + 阴影偏 Kodak 500T 青绿"

### 2.2 空间心理学
每一个空间在心理上对人物做了什么。

- 低天花板 = 压迫（不是"天花板很矮"，是"天花板低到人物站起来头顶距天花板不到 30cm"）
- 大窗户 = 暴露（外面可以看进来）
- 长走廊 = 不可避免的接近（越走越近，无处可逃）
- 角落 = 安全或被困（取决于人物状态）
- 门 = 选择或威胁（开着的门是可能性，关着的门是秘密，半开的门是不确定）

**实现方式:**
- 场景资产板中标注空间心理学——不是只标"这里是卧室"，是"卧室角落=她的安全区，衣柜=威胁源，镜子=自我审视"
- 故事板俯视动线图中标注空间的"热区"和"冷区"——人物在哪里有力量，在哪里是脆弱的

### 2.3 场景中的叙事物件
每一个在画面中停留超过 1 秒的物体，观众会赋予它意义。所以要控制它们。

- 第一类——推动剧情: 钥匙、水杯、衣柜门（必须有、必须一致）
- 第二类——建立人物: 书架上某本书、冰箱上的便利贴、床头的照片（不说话，但说很多）
- 第三类——建立世界: 空调外机、消防梯、地面积水（说"这是真实的地方，不是 AI 生成的"）
- 禁止类——随机出现的物体、位置漂移的物体、没有叙事意义的装饰性物体

**实现方式:**
- 场景资产板中分类标注四种物件
- 故事板中标注哪些物件在当前镜头中可见及其状态

### 2.4 材质语言
材质不是装饰——材质是触觉记忆。

- 磨损的木地板 = 时间、历史、被踩过的生活
- 冰冷的金属 = 工业、疏离、不近人情
- 起球的羊毛大衣 = 穿了很久、经济状况、不在意体面或无力在意
- 有冷凝水珠的玻璃杯 = 在这里放了至少二十分钟了

**实现方式:**
- 资产板中标注关键材质和老化/磨损状态
- Seedance prompt 中在描述物件时带材质细节（不是"一个杯子"，是"杯壁外侧有三颗冷凝水珠正缓慢下滑的透明玻璃杯"）

## 三、摄影 / Cinematography

### 3.1 画幅选择的心理含义
- 16:9: 当代标准，中性。适合大多数叙事。
- 2.35:1 / 2.39:1: 电影宽银幕。史诗感、空间感、人物在环境中的渺小。
- 4:3: 复古、私密、幽闭。人物被框住。适合心理惊悚、角色研究。
- 1:1: 正方形。当代艺术感、社交媒体原生。构图被限制，被迫简洁。
- 9:16: 竖屏。人物占据主导，环境为陪衬。适合人物驱动的短片、UGC 感广告。

### 3.2 镜头选择的心理含义
不是"85mm 拍人像"——是"85mm 把人物从环境中剥离，让观众只看这个人"。

- 24mm 广角: 把人物放进环境。人物小，世界大。孤独、渺小、暴露。
- 35mm: 人眼自然视角。让人物和环境保护关系。纪录片感、真实感。
- 50mm: 接近人眼注视时的视角。注意力开始集中在人物上，环境退后。
- 85mm: 肖像镜头。环境被压缩和虚化。观众只看这个人——她的毛孔、她的微表情、她的孤独。
- 135mm+: 极度压缩空间。人物仿佛被贴在背景上，无处可逃。压迫感。
- 变焦 vs 定焦: 变焦 = 观察者的目光选择性放大。定焦 = 摄影机移动带来的视角变化是真实的。

**镜头特性（不只是焦段）:**
- 老镜头: 低反差、眩光、像场弯曲。柔和的真实，不是锐利的虚假。
- 现代镜头: 高反差、高锐度、干净。但容易"太干净"——AI 味。
- 变形宽银幕: 椭圆焦外、水平眩光。电影感的第一联想。
- 球面镜头: 正常焦外。自然、不刻意。

**实现方式:**
- 每个 Seedance shot 头标注焦段 + 镜头特性关键词（如"35mm 老镜头低反差"而非只"35mm"）
- 如果用浅景深，说清楚焦点落在哪里、为什么落在这里

### 3.3 景深作为叙事工具
- 深焦（f/8-f/16）: 人物和背景都清晰。人物没有隐私——环境在说话，环境在指控。
- 中焦（f/2.8-f/5.6）: 人物清晰，背景可辨识但不抢戏。叙事平衡。
- 浅焦（f/1.2-f/2）: 只有人物（或人物的一部分）清晰。完全聚焦内心。但过度使用 = 广告感。
- 移焦: 从一个焦点移到另一个焦点。观众被强制转移注意力。揭示或隐瞒。

### 3.4 构图语法
每一个取景决定都在说一句话。

- 中心构图: 稳定、正式、被审视。神像、证件照、审判席。
- 三分法偏移: 自然、动感、有呼吸空间。人物看向的一侧留空间——否则是压迫。
- 极端偏移: 人物在画面边缘——他们被逼到了角落，或者世界比他们大太多。
- 低角度: 人物有力量、威胁、或渺小中的尊严。
- 高角度: 人物脆弱、被俯视、被控制。上帝视角在看一个无助的人。
- 荷兰角: 世界是歪的。潜意识不安。用一次就够了，同一部片不超过两次。
- 过肩: 我们和说话者是一边的。我们看着对方。对话感。
- 单人正面: 我们面对这个人。没有遮掩。她在和我们说话——或在被我们审视。

**实现方式:**
- 故事板镜头表中每个 shot 标注构图意图（不是"中景"，是"中景/三分法/人物偏右、左侧留呼吸空间"）
- Seedance prompt 描述中融入构图动机

### 3.5 摄影机运动语言
摄影机为什么动？

- 固定（locked-off）: 不动。世界在运转，摄影机只是记录者。适合——观察、等待、紧张、仪式感。人物在框内表演，摄影机不帮忙。
- 手持: 有人在这里。有呼吸。有体温。适合——代入、纪实、不安、紧迫。手持不是乱晃——手持是有节奏的。
- 斯坦尼康/稳定器: 飘浮。灵魂在空间中滑行。适合——跟随、优雅、疏离的观察。但极容易 AI 味——飘浮 = 没有动机。
- 推近（push-in）: 正在接近一个真相。缓慢推 = 不可抗拒。快速推 = 突然的威胁。
- 拉远（pull-out）: 正在离开、失去、放手。缓慢拉 = 渐行渐远。快速拉 = 突然的揭露（原来旁边还有东西）。
- 横移（tracking）: 平行跟随。陪伴或追踪。侧跟 = 观察。后跟 = 追逐或护送。
- 摇摄（pan）: 眼睛在转。从 A 看到 B。快摇 = 突然的注意。慢摇 = 犹豫地看。
- 升降（boom/crane）: 从低到高 = 从参与到俯视。从高到低 = 从天降到卷入。

**AI 味运镜禁止清单:**
- 无动机环绕（为什么绕着人物转？没有理由 = AI 在炫耀）
- 无动机飘浮（没有呼吸节奏的稳定器 = 幽灵视角 = AI 味）
- 无动机变焦推拉（同时变焦又推轨 = 希区柯克式，只在特定心理时刻用）
- 无动机慢动作（慢动作是强调，强调需要理由）

### 3.6 胶片/数字色彩科学
每一部电影有一卷"胶片"——决定了它的视觉基因。

- Kodak 500T (5219): 阴影青绿偏移，高光保留暖色，颗粒在中灰区域最明显。适合——暗调故事、心理惊悚、城市夜晚。
- Kodak 250D (5207): 日光平衡，暖肤色，低颗粒，高宽容度。适合——自然主义、阳光、白天的真实。
- Kodak Ektachrome 100D: 正片/反转片，高饱和，蓝调倾向，极低颗粒但极高反差。适合——时尚、广告、怀旧 70s。
- Fuji Eterna Vivid 160T: 青色高光，品红暗部，日系电影感。适合——霓虹、科幻、都市孤独。
- ARRI Alexa LogC → Rec709: 现代数字电影感。平滑高光滚降，暗部保留细节但不染色。
- Sony Venice S-Gamut3: 更冷的肤色，更高的分辨率感。适合——干净的当代叙事。

**实现方式:**
- 项目设置中声明"本片色彩科学: Kodak 500T (5219) 特性——阴影青绿偏移，高光暖色保留，中灰颗粒"
- Seedance prompt 中需要色彩质感描述时引用已声明的色彩科学基准

## 四、灯光 / Lighting (Gaffer)

### 4.1 光质
不是"亮"或"暗"。是光的性格。

- 硬光: 清晰阴影边缘，高对比。中午日光、聚光灯、裸灯泡。权威、暴力、真相、不掩饰。
- 柔光: 渐变阴影，低对比。阴天、柔光箱、窗户透过薄纱。温柔、暧昧、回忆、不残忍。
- 漫射光: 无阴影，全方向。雾天、大柔光罩、穹顶光。无性格、梦境、死亡、空白。

### 4.2 光位
光从哪个方向来，决定了我们看到什么、漏掉什么。

- 正面光: 扁平，无阴影，直面观众。偶像、审讯、没有秘密。
- 侧光: 半张脸在光里半张脸在暗里。分裂、双重人格、内心矛盾。
- 逆光: 人物是剪影或在光圈中。神秘、神圣、即将揭晓。
- 顶光: 眼眶深暗，颧骨突出。疲惫、邪恶、非人。恐怖片的默认光位。
- 底光: 面部结构反转。鬼脸、非自然、超现实。通常只用一次。
- 实用光源（practical）: 画面内可见的光源。台灯、电视、窗户、蜡烛。光有来源——真实。

### 4.3 光比
- 高光比（Key:Fill = 4:1 或更高）: 戏剧性、危险、电影感。阴影是另一个角色。
- 中光比（Key:Fill = 2:1）: 自然、真实、纪录片。有体积但不夸张。
- 低光比（Key:Fill = 1:1 或极低）: 扁平、梦境、抑郁、空虚。
- 剪影（Key:Fill = ∞）: 人物是形状而不是面孔。神秘、匿名、即将揭晓。

### 4.4 色温动机
- 2700K-3200K（暖/钨丝）: 室内、亲密、安全或窒息
- 4300K-5600K（中性/日光）: 室外、真实、客观
- 6000K-8000K（冷/阴天）: 疏离、科技、不安
- 混合色温: 暖室内 + 冷窗外 = 两个世界在一个人身上交锋

**实现方式:**
- 故事板灯光区域标注主光方向、光质、光比、色温
- Seedance prompt 描述中融入光位和光比的叙事动机——不是"右侧来光"，是"窗外街灯从右侧穿过百叶窗在她脸上切出狱栏般的条纹"

## 五、录音 / Sound

这是 AI 视频最容易被忽视的维度。但声音占电影体验的 50%。

### 5.1 声音层次
一段专业的声音有至少四层:

1. **房间音 / Room Tone**: 空间自己的呼吸。每个房间有不同的频率——小房间高频反射多，大空间低频混响长。没有房间音 = 真空 = AI 味。
2. **环境音 / Ambience**: 空间外的世界。窗外车流、远处人声、风声、雨声。环境的"在场"让空间真实。
3. **拟音 / Foley**: 人物动作的声音。脚步（什么鞋、什么地面、什么重量）、衣物摩擦（什么面料）、手触碰物体（什么材质）。
4. **对白 / Dialogue**: 语言。以及语言之间的沉默。以及沉默中能听到的呼吸。

### 5.2 声音的叙事功能
- 声音先于画面（J-cut）: 下一个场景的声音在画面切换前就进入了。耳朵先到达，眼睛随后。平滑、预示、连接。
- 画面先于声音（L-cut）: 上一个场景的声音延续到下一个画面。余韵、不舍、惯性。
- 声音桥: 同一个声音跨越两个镜头，连接它们。
- 声音突然消失: 比爆炸更响的是爆炸后的耳鸣。比恐惧更恐惧的是恐惧时的寂静。
- 声音低频（<100Hz）: 不祥、压迫。地震、巨物、恐惧。不需要看到声源——身体先感觉到了。
- 声音高频（>8kHz）: 刺痛、警觉、耳鸣。枪响后、冲击后、极端的意识状态。
- 声场收窄: 进入人物的主观听觉。环境声退后——只剩心跳、呼吸、一个声音。私密空间建立。

### 5.3 Seedance 的声音限制
- Seedance 生成的声音有限。不能期待精确对白同步或复杂音乐
- 但可以期待: 环境音层次、基本脚步拟音、呼吸声、氛围低频
- 精修需要在后期（替换音乐、补录对白、调整混音）
- 所以提示词中的声音描述重点在"给后期留接口"——说清楚需要什么、在哪里

### 5.4 声音设计的关键时刻
- 安静: 不是没有声音。是房间音 + 远处细微环境 + 人物呼吸。安静是有质感的。
- 渐强: 声音从小到大。紧张累积。但不要滥用到每一段都有。
- 渐弱: 声音从大到小。情绪消退、退场、离开。
- 突然: 一声。然后余响。然后安静。

**实现方式:**
- 故事板声效标注区域按四层声音标注（房间音/环境/拟音/对白意图）
- Seedance prompt 描述声音时给出具体参数: 频率范围、dB 相对值、空间感（干/湿）、进入和退出方式
- 标注"后期需补"的声音（精确对白、音乐、复杂音效）

## 六、表演 / Performance

### 6.1 表演的层级
- 第一层——动作: 做什么。走路、坐下、拿东西。
- 第二层——表情: 脸上有什么。微笑、皱眉、哭泣。
- 第三层——微表情: 脸上有什么但试图隐藏。0.5mm 的嘴角上扬在被压制。这个层级是电影和 AI 视频的分界线。
- 第四层——身体语言: 身体在说什么，可能和表情矛盾。嘴上在笑，脚已经在向门的方向了。
- 第五层——呼吸: 自主神经系统的窗户。呼吸不会说谎。这是表演中最强大的工具，也是最容易被忽略的。

### 6.2 表演指导语言
不要说"悲伤"。说:

"她的下巴比平时低了 1cm。不是低头——是下巴的重量增加了。眼睑也比平时低——不是闭眼，是睁着眼但上眼睑只抬到平时 70% 的高度。呼吸——鼻吸比平时浅，呼出时有极轻微的气息声——不是叹气，是身体自己在释放多余的紧张。"

**关键词:** 具体肌肉、具体幅度、具体时间、具体节奏。

### 6.3 静止作为表演
静止不是"没有动作"。静止是"所有微小的动作都被抑制了"。

- 完全静止: 恐惧的顶峰。捕食者面前。身体说"不要被发现"。
- 有呼吸的静止: 沉思、等待、忍耐。身体在最低功耗运转。
- 僵硬: 不自然的静止。在控制什么。在撒谎。在害怕。

### 6.4 反应
表演不只是动作——动作之后的反应往往比动作本身更揭示人物。

- 听到一个声音 → 先停住呼吸 → 然后转头 → 然后眼神定住 → 然后决定怎么做
- 被说了一句伤人的话 → 先什么表情都没有（延迟 0.3s）→ 然后微表情（下唇收紧）→ 然后恢复（比平时更快的眨眼）→ 然后反应（微笑或回击）
- 看到不该看到的东西 → 瞳孔先有反应（放大）→ 然后身体后倾 → 然后面部跟上

**实现方式:**
- 每个关键情绪时刻给出完整的反应链——不是"她害怕了"，是"她先停住呼吸 0.8s→然后重心后移 2cm→然后下唇收紧→然后瞳孔放大"
- Seedance prompt 中反应链的时间精度在 0.1s 级别

## 七、剪辑 / Editing

Seedance 不直接做剪辑，但故事板决定了剪辑的骨架。每个镜头之间的连接点在故事板上就已经决定了。

### 7.1 剪辑点的类型
- 硬切: 瞬间切换。直接、诚实、不加修饰。适合——大多数叙事场景。
- 跳切: 在同一机位上剪切时间的跳跃部分。焦虑、不耐烦、风格化。
- 匹配剪辑: 两个镜头之间有形状、动作或构图的呼应。优雅、巧思、满足。
- 叠化: 两个画面短暂重叠。时间流逝、记忆、两个世界的交融。
- 淡入/淡出: 开始或结束一个章节。呼吸。停顿。

### 7.2 剪辑节奏
- 动作场景: 镜头越来越短。开始时 2-3s 一个镜头，高潮时 0.5-1s。
- 对话场景: 节奏跟随呼吸。紧张时越来越快，释放时慢下来。
- 悬疑场景: 镜头越来越长。不给观众喘息——让他们盯着一个画面直到不安。
- 蒙太奇: 镜头短、密度高、信息量大。用剪辑创造新的含义。

### 7.3 180 度法则与轴线
- 两个人物对话——他们之间有一条无形的轴线。摄影机不能越过这条线——否则观众会困惑谁在看向哪里。
- 有意打破 180 度: 混乱、权力转移、世界观崩塌。
- 视线匹配: 人物 A 看屏幕右方 → 下一个镜头是人物 B 在屏幕左方看回来 → 他们建立了眼神接触。

**实现方式:**
- 故事板镜头表中每两个相邻镜头之间标注剪辑类型（CUT/DISSOLVE/MATCH CUT）
- 多人物场景故事板俯视动线图中标注 180 度轴线
- Seedance prompt 中描述段内和段间的节奏设计意图

## 八、服化道 / HMU (Hair, Makeup, Wardrobe)

### 8.1 服装作为叙事
服装不是在说"这个角色穿什么"——是在说"这个角色是谁，今天经历了什么，现在是什么状态"。

- 刚下班: 衬衫有坐痕，领口松了，袖口微皱。不是"穿衬衫"，是"衬衫已经被穿了一天"。
- 刚淋雨: 湿不是均匀的——肩头最湿，发梢滴水，裤脚溅了泥。不是"湿了"，是"湿在哪里、湿了多少、湿了多久"。
- 紧张时: 领口被拉松——她无意识地扯了领口。袖口被卷起又放下——反复。
- 奔跑后: 衣领歪了，下摆从裤腰里跑出来一截，鞋带松了。

### 8.2 服装和光线的关系
不同面料在同样的光下完全不同。
- 羊毛: 吸光，只有轮廓高光
- 丝绸: 每一步都在改变高光位置
- 化纤: 锐利的高光点，不自然
- 棉: 漫反射，均匀的亮

### 8.3 发型作为状态
- 刚下班: 早上扎的马尾松了，前额碎发多了，有一缕从耳后滑出来了
- 淋了雨: 湿发贴在额头和颧骨上，发梢结成一缕一缕的
- 恐惧: 头发没有变——但她拨头发的次数多了。或者完全不拨了——静止。
- 奔跑后: 马尾歪了，发圈快掉了

### 8.4 妆容
- 日常: 有粉底的痕迹但已经在 T 区被皮肤油脂融合。唇膏在杯沿和午餐后已褪。
- 哭泣: 不是"眼线花了"——是先眼眶红了，然后眼泪在下睫毛聚集，然后第一滴落下时带走了一点粉底，留下一道比周边皮肤亮一度的痕迹。
- 恐惧: 面无血色——不是变白，是毛细血管收缩导致皮肤下层的红色减少，露出皮肤本身偏黄或偏橄榄的底色。

**实现方式:**
- 角色资产板中标注服装面料和光交互、发型基准和可变状态、妆容状态变化规则
- Seedance prompt 中状态变化的描述精确到部位和程度，不是"衣服皱了"是"衬衫下摆从西裤里跑出了约 3cm"

## 九、VFX / 物理特效

### 9.1 什么时候用 VFX
- 无法实际拍摄的环境（太危险、太贵、不存在）
- 增强现实——让真实的东西更有表现力（不是替代，是增强）
- 过渡和转场——连接两个无法直接连接的镜头

### 9.2 物理特效优先
能实际拍到的就不要 VFX:
- 雨: 能实拍就实拍。VFX 的雨大概率均匀得像 shower。
- 烟/雾: 实拍的烟有自己的物理行为——上升、扩散、被风拉扯。VFX 的烟雾容易像均匀的图层。
- 血: 实拍的血有重量、黏度、在不同表面的扩散方式。VFX 的血容易太亮或太均匀。

### 9.3 VFX 的 AI 味问题
- 粒子太均匀 = AI 味。自然的粒子是不均匀的——大小、速度、密度都有分布。
- 光晕太完美 = AI 味。镜头光晕有瑕疵——不是每盏灯都有光晕，光晕的形状因镜头而异。
- 魔法/VFX 特效 = 给一个物理来源。不是凭空发光——是从一个物体/一个动作中触发。

**实现方式:**
- 尽量避免让 Seedance 生成复杂 VFX
- 如果必须 VFX，给明确的物理参考——不是"魔法光"，是"从她指尖出发的细光线，像一根发光的线在空气中游动，速度不均匀，有微弱的残影衰减"
- 标注"后期增强"的 VFX 需求

## 十、节奏与呼吸 / Rhythm & Breathing

这不是一个技术维度——这是所有技术维度的总指挥。

### 10.1 一场戏的呼吸
- 不能从头紧张到尾——观众会疲劳
- 不能从头舒缓到尾——观众会无聊
- 紧张→微释→更紧张→崩溃→寂静。这才是呼吸。
- 在最高紧张之后需要一个释放——不一定是对角色的释放，可以是对观众的。"让她安全了 0.5 秒，然后再拿走。"

### 10.2 一段 15 秒内的呼吸
即使是 15 秒的单段，也需要内在的节奏:
- 不是"5-6 个镜头均匀分布"
- S01: 4s 慢入（呼吸的"吸"）
- S02: 2s 顿点（呼吸的"短暂停"）
- S03: 6s 核心（呼吸的"呼"——这口气最长，因为是内容核心）
- S04: 3s 余韵（呼吸的"停"——呼出后气息悬空的瞬间）
- 或者: 就 1 个长镜头 15 秒——从头到尾一口气。那是另一种呼吸。

### 10.3 段与段之间的呼吸
- 上一段的末帧不能是"动作中间"——必须是"一个呼吸的末尾"
- 下一段的起帧必须是"下一个呼吸的开始"
- 段间手递不是"接上"，是"换气"

### 10.4 声音的呼吸
- 不能全程有背景音——声音也需要喘息
- 安静过后，第一个声音的出现是事件
- 连续的强音之后突然寂静——比任何声音都响

## 母版检查清单 / Master Checklist

在交付任何 prompt 文件之前，按此清单逐项对照。每一项都必须被满足——要么在设计中有体现，要么明确声明"本段不适用"并有合理理由。

**Pre-Production:**
- [ ] 项目圣经是否已完成？（多段/长片项目）
- [ ] 输入路由是否正确分类？
- [ ] 模型能力边界是否已检查？（这场戏 Seedance 能做到吗？）
- [ ] 色彩宪法是否已声明？（主色/辅色/强调色/中性色/禁止色）
- [ ] 胶片/色彩科学基准是否已声明？

**Art Department:**
- [ ] 角色资产板：面部锚点/微表情范围/服装面料与光交互/发型状态变化规则/妆容状态变化规则/心理档案/表演基准
- [ ] 场景资产板：多角度空间/光源位置与色温/固定物位置/四种叙事物件分类/材质语言/空间心理学标注
- [ ] 道具资产板：多视图/材质/尺寸/持握方式/故事中状态变化/声音特征

**Cinematography:**
- [ ] 画幅选择是否有叙事理由？
- [ ] 每个 shot 的焦段是否有心理含义？
- [ ] 景深是否有叙事动机？（焦点落在哪里、为什么）
- [ ] 构图是否有语法？（中心/三分/极端/低角/高角/荷兰/过肩）
- [ ] 摄影机运动是否有动机？（锁/手持/推/拉/移/摇/升降）
- [ ] 是否包含了镜头特性？（老镜头低反差/变形宽银幕/球面）
- [ ] AI 味运镜是否已排除？（无动机环绕/飘浮/变焦推轨/滥用慢动作）

**Lighting:**
- [ ] 每个 shot 的主光方向/光质/光比/色温是否明确？
- [ ] 光源是否有物理来源？（实用光源 or 明确的环境光）
- [ ] 光是否有叙事动机？（不是"照亮"，是"揭示"或"隐藏"）

**Sound:**
- [ ] 四层声音（房间音/环境/拟音/对白意图）是否都已标注？
- [ ] 声音的进入和退出方式是否明确？
- [ ] 关键声音时刻是否有精确时间点？
- [ ] 后期需补的声音是否已标注？
- [ ] 是否有"安静"的时刻？安静的质感是否已定义？

**Performance:**
- [ ] 每个关键情绪时刻是否有完整的反应链（动作→微表情→身体语言→呼吸）？
- [ ] 微表情描述是否精确到肌肉和幅度？
- [ ] 呼吸变化是否有频率和深度描述？
- [ ] 静止的时刻是否有定义？（是哪种静止、为什么静止）

**Editing:**
- [ ] 相邻镜头之间是否标注了剪辑类型？
- [ ] 180 度轴线是否已检查？（多人物场景）
- [ ] 段内节奏是否有呼吸？（吸→停→呼→悬）
- [ ] 段间手递是否在"一个呼吸的末尾"和"下一个呼吸的开始"？

**HMU:**
- [ ] 服装状态变化是否按时间线标注？
- [ ] 发型变化是否按情绪/动作状态标注？
- [ ] 妆容变化是否有触发条件和具体表现？

**VFX:**
- [ ] 是否有 Seedance 无法生成、需要后期增强的 VFX 需求？
- [ ] 任何 VFX 是否给定了物理参考？

**Final Check:**
- [ ] Seedance prompt ≤ 2000 字符？
- [ ] 故事板文字是否为最小（只有逻辑标签）？
- [ ] 故事板和 Seedance prompt 之间是否有重复内容？
- [ ] 禁止漂移列表是否覆盖了所有需要锁定的元素？

# Sixteen-Panel Dance Choreography Board Pattern

Use this pattern for dance, martial movement, ritual performance, fashion movement, cultural performance, or any sequence where body motion must be controlled through many clear action nodes.

## Best For

- dance videos
- cultural dance
- water dance / veil dance / sleeve dance
- martial arts forms
- stylized action choreography
- runway/fashion motion
- graceful one-performer movement sequences
- Seedance clips where pose continuity and fabric/water/reflection motion matter

## Core Intent

Create a 4x4 sixteen-panel storyboard that breaks one complete performance into 16 key motion nodes.

Each panel must be:

- visually distinct
- sequential
- physically possible
- readable as a camera shot
- useful as Seedance reference

## Layout

Use:

- 4x4 grid
- panel numbers 1-16
- short action title per panel
- optional one-line description per panel
- light motion arrows or trajectory lines
- clean high-resolution visuals
- consistent performer identity, costume, location, lighting, and style

Avoid overcrowding. The board should be clear enough for video generation.

## Required Design Fields

### Overall Positioning

State what the board is:

- not a photo set
- not a generic dance tutorial
- a cinematic choreography storyboard
- a complete performance split into 16 action nodes

### Scene

Describe a believable performance space:

- real location, not empty studio unless requested
- time of day
- light source
- atmosphere
- ground/water/reflection/fabric interaction if relevant

### Performer

Lock:

- same performer identity
- same face
- same costume system
- same temperament
- same physical ability level

### Costume / Fabric Motion

Specify how fabric participates:

- veil
- skirt
- sleeves
- ribbons
- jewelry
- hair ornaments
- water/reflection if relevant

### Motion Sequence

Use 16 specific beats:

- opening pose
- initiation
- reach/sweep
- expansion
- traveling step
- low movement
- rise
- turn
- extension
- spin start
- spin continuation
- leap or travel
- kneel / pause / glance
- lift / hold
- climax move
- final pose

Adapt these to the project.

## Seedance Pairing Prompt Structure

After generating the 16-panel board, Seedance prompt should include:

- strictly follow panels 1-16 in order
- each panel is a clear action node and camera node
- same performer, same face, same costume
- realistic physically possible movement
- no static posing
- fabric/hair/water/reflection responds to motion
- camera stays elegant and stable
- most shots show full body when motion clarity matters
- no random cuts or chaotic camera

## Reusable Image2 Instruction Block

```text
请创作一张高完成度、超高清、清晰通透的 4×4 十六宫格电影动作/舞蹈分镜图。这不是普通写真，也不是简单教学图，而是一张完整表演被拆成 16 个关键动作节点的电影分镜海报。每一格既是独立镜头画面，又必须按 1-16 顺序形成连续流动。

版式要求：4×4 十六宫格，每格有明显编号 1-16、简短动作标题、可选一句说明，可加入轻量动作箭头或轨迹线。整体清晰、工整、信息完整但不要拥挤。人物、服装、动作、场景、地面/水面/倒影/道具都必须清楚可见。

人物要求：全程保持同一人物身份、同一张脸、同一气质、同一套服装体系。每一格动作明显不同，动作幅度要大，但必须是真人/角色真实可完成的身体动作，不要悬浮，不要魔法化，不要只是摆拍。
```

## Reusable Seedance Instruction Block

```text
请根据我提供的 16 宫格参考图生成视频。严格按照画面 1 到画面 16 的顺序依次生成。每个画面都是明确的动作节点和镜头节点。全程保持同一人物、同一张脸、同一套造型、同一场景和同一光线。动作必须流畅、连续、真实可完成，不能只是摆姿势。服装、纱、袖、头发、水面、倒影或相关道具都要随着动作自然响应。

镜头要稳定、流动、优雅，可以使用缓慢推进、横向跟拍、小幅环绕、低机位、短暂停顿和高潮段落环绕。不要乱晃，不要频繁快切。大部分时间完整展示全身，确保动作线条和服装运动清楚可见。
```

## When Not To Use

Avoid this pattern for:

- dialogue scenes
- product process videos
- complex multi-character blocking
- long narrative scenes with multiple locations

For product/process videos, use `infographic-8-shot-poster.md`.

For drama/action scenes, use `storyboard-control-board.md` or `two-part-cinematic-production-board.md`.

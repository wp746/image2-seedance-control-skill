# Infographic 8-Shot Storyboard Poster Pattern

Use this pattern for clean, fast, satisfying 12-15 second sequences where Seedance 2.0 needs a simple storyboard poster to follow.

## Best For

- product ads
- food and drink making
- craft/process videos
- warm lifestyle shorts
- tutorial-like visual sequences
- social ads
- mascot/product demos
- simple one-character workflows

## Core Intent

Create a crisp infographic storyboard poster that Seedance can read as a visual shot list.

This pattern works because it combines:

- a clear top header
- total video time
- exact shot count
- rhythm keywords
- legend icons
- one consistent character/location note
- 8 short shot descriptions
- footer with flow, camera tips, light/style, and character notes

## Layout

Use:

- wide 16:9 layout
- white or very clean background
- black borders
- bold readable typography
- 8 panels
- warm, clear rendering style when appropriate
- consistent character and setting across all panels

Top header should include:

- project title
- total video time
- shot count and rhythm, e.g. `8 SHOTS · FAST · WARM · SATISFYING`
- legend icons, e.g. `ACTION`, `HEAT`, `TIME HINT`, `INGREDIENT`

Panel content:

- each panel shows one action beat
- each panel has a short caption
- use consistent subject and setting
- use strong visual progression from setup to payoff

Footer should include:

- `VIDEO FLOW`: e.g. `8 shots × ~1.5s = 12-15 seconds`
- `CAMERA TIPS`: close-ups, 45-degree angles, wide shots, reaction shots
- `LIGHT & STYLE`: color palette, depth of field, rendering style
- `CHARACTER/PRODUCT NOTES`: one subject, one process, one emotional payoff

## Shot Rhythm

Default rhythm:

- 8 shots
- each shot around 1.5 seconds
- total 12-15 seconds

Typical structure:

1. open/setup
2. ingredient/object enters
3. hand/action close-up
4. transformation/process highlight
5. heat/energy/motion beat
6. beauty payoff
7. delivery/reveal
8. reaction/emotional payoff

## Reusable Instruction Block

```text
创建一张简洁明快的信息图式故事板海报。16:9 横屏，干净白色背景，黑色边框，醒目的黑色字体，8 个清晰分镜面板。顶部必须包含片名、总时长、镜头数量、节奏关键词和图例图标。主体始终保持同一个角色/产品/场景。每个面板只表现一个明确动作节拍，并配短标签。底部必须包含 VIDEO FLOW、CAMERA TIPS、LIGHT & STYLE、CHARACTER/PRODUCT NOTES。整体要让 Seedance 2.0 一眼读懂视频顺序、动作、镜头重点和情绪回报。
```

## Seedance Pairing Prompt

After generating the poster, the user can upload it to Seedance and say:

```text
Use the attached storyboard image as the exact reference. Create a 12-15 second 16:9 video that follows the 8-shot storyboard exactly. Preserve the same character/product, setting, color palette, lighting style, action order, and emotional payoff. Keep each beat fast, clear, and satisfying.
```

## When Not To Use

Avoid this pattern for:

- complex drama scenes
- subtle dialogue
- multi-character blocking
- long action choreography
- horror suspense requiring slow pacing
- scenes needing precise continuity across many locations

For those, use `storyboard-control-board.md` or a more specialized storyboard branch.

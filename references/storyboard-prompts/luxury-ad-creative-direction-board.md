# Luxury Ad Creative Direction Board Pattern

Use this pattern for 15-20 second broadcast-quality luxury commercials built from one ultra-detailed creative direction board.

## Best For

- luxury fragrance ads
- electric car ads
- alpine lodge / travel hospitality ads
- skincare / beauty ads
- fashion campaigns
- premium product films
- glossy brand spots where mood, styling, transitions, and hero shots matter

## Core Method

Create one detailed creative direction board / visual planning sheet in Image2, then use it as the reference for Seedance.

The board should function like a director's pre-production guide:

- shared creative direction
- character/styling reference
- environment/set design
- route/camera position guide
- 8-scene storyboard
- lighting/mood/style notes
- mood keywords
- audio/tone
- cinematography notes
- concise Seedance animation prompt

## Board Structure

### Top Header: Shared Creative Direction

Include icon-and-text items:

- scene count
- ad category
- location
- time-of-day arc
- palette
- visual arc

Example visual arc:

```text
ARRIVAL -> SEDUCTION -> INTIMACY -> BRAND REVEAL
```

### ChatGPT Image Prompt Block

Include a short concise image prompt that defines the campaign mood.

Keep it readable. Do not overfill the board with text.

### Character + Styling Reference

Show the same model/hero subject from multiple views:

- front stand
- side view
- back view
- walking pose
- close-up portrait

Below include wardrobe/accessory/product thumbnails:

- fabric
- hero product
- jewelry/accessories
- shoes
- environmental motif

Add a short consistency checklist.

### Environment + Set Design

Include:

- large scenic hero image
- location atmosphere
- reflective surfaces / lighting / weather / architecture
- route guide or loose floor plan
- camera positions 1-8
- route zones, e.g. entrance, bar, balcony, skyline edge

### Storyboard - 8 Scenes

Use 8 numbered frames.

Each frame should include:

- image
- lens feel
- movement
- action
- emotion

Typical structure:

1. wide location reveal
2. subject enters / walks through setting
3. over-shoulder or environment connection
4. product use / sensory action
5. macro product close-up
6. profile / emotional pause
7. close-up expression / wind / texture
8. wide hero shot with product/brand reveal

### Bottom Strip

Include:

- LIGHTING / MOOD / STYLE NOTES
- 5 small reference thumbnails and captions
- MOOD + KEYWORDS
- AUDIO / TONE
- CINEMATOGRAPHY NOTES
- SEEDANCE 2.0 ANIMATION PROMPT

## Reusable Image2 Instruction Block

```text
请创建一张电影级广告创意指导板 / visual planning sheet，用于 15-20 秒高端品牌广告。它必须像导演的前期制作指南，清晰易读、设计专业、编辑感高级，而不是普通海报。

整体使用优雅的 editorial style、clean grid system、premium typography 和清晰面板结构。顶部标题为 SHARED CREATIVE DIRECTION，包含：场景数量、广告类型、地点、时间变化、色彩体系、视觉弧线。

必须包含：CHATGPT IMAGE PROMPT 简短文本块、Character + Styling Reference、Wardrobe + Accessories 缩略图、Environment + Set Design 大图、LOCATION + ROUTE GUIDE 俯视路线图、Camera Positions 1-8、Storyboard – 8 Scenes、LIGHTING / MOOD / STYLE NOTES、MOOD + KEYWORDS、AUDIO / TONE、CINEMATOGRAPHY NOTES、SEEDANCE 2.0 ANIMATION PROMPT。

Storyboard 的 8 个画面每格都要有 Lens Feel、Movement、Action、Emotion。整体要适合后续直接作为 Seedance 2.0 的广告视频参考图。
```

## Seedance Prompt Style

The board may contain a concise Seedance prompt like:

```text
Slow cinematic dolly and glide shots across [location/time]. Follow [subject] as [wardrobe/material/product] moves gently. [motif] drifts through frame, [product effect] catches light, background lights bloom into soft bokeh. Mix aerial reveal, tracking walk, intimate product close-ups, and final hero reveal. Smooth movement, shallow depth of field, glossy luxury finish.
```

## Category Adaptation

### Luxury Fragrance

Use:

- glamorous model
- hero perfume bottle
- fabric/hair wind
- petals/mist
- city lights/bokeh
- intimate close-ups
- final brand reveal

### Electric Sports Sedan

Use:

- coastal dawn route
- vehicle hero angles
- wheel/mirror/headlight close-ups
- road motion
- sunrise reflections
- final driving hero shot

### Alpine Eco-Lodge / Travel

Use:

- sunrise mountains
- lodge environment
- arrival route
- interior warmth
- nature texture
- slow luxury travel pacing

### Skincare / Beauty

Use:

- oceanfront sanctuary
- skin texture
- product macro
- water/light reflections
- soft ritual motion
- clean final packshot

## When Not To Use

Avoid this pattern for:

- non-commercial narrative drama
- complex dialogue scenes
- long episodes
- raw action choreography

Use `smart-shot-directing-sheet.md` for narrative cinematic scenes.
Use `infographic-8-shot-poster.md` for simpler product/process boards.

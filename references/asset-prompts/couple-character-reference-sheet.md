# Couple Character Reference Sheet Pattern

Use this pattern when designing two romantic characters as one shared visual bible for Seedance 2.0.

## Best For

- romantic couple videos
- AI drama couples
- relationship introductions
- dating show / short drama intros
- two-character emotional scenes
- any project where chemistry and shared identity matter

## Core Intent

Create a unified couple character reference sheet, not two separate character sheets.

The board should emphasize:

- romantic chemistry
- shared identity
- visual harmony
- height/scale relation
- emotional continuity
- interaction potential

## Layout

Use a clean 4:3 horizontal layout on a pure white or neutral background.

Keep the board:

- minimal
- technical
- premium
- uncluttered
- clearly labeled

Apply the requested style only to the characters, not to the board design.

## Required Modules

### 1. Cute Couple Info Card

Two soft profile cards side by side with a small connector.

Each card includes:

- portrait
- name
- alias
- role/dynamic
- age
- personality
- core theme
- speech accent
- couple vibe

### 2. Small Color Palette

Use five small heart-shaped color swatches.

Keep it secondary.

### 3. Main Identity + Scale Sheet

Dominant section.

Show both characters together:

- front
- 3/4 view
- side
- back

Use neutral but connected poses.

Include measurement guide lines and height marks.

### 4. Romantic Expression Progression

Exactly 8 panels:

- Neutral Warmth
- Shy Smile
- Playful Affection
- Loving Gaze
- Gentle Laughter
- Tender Concern
- Emotional Closeness
- Relieved Togetherness

### 5. Chemistry Moments

Exactly 5 panels:

- soft eye contact
- shared smile
- shy glance
- leaning closer
- quiet emotional pause

### 6. Head Detail Sheet

Show both characters from:

- 3/4 headshot
- side headshot
- low angle
- diagonal angle

### 7. Close-Up Couple Pose

One larger cinematic chest-up pose with strong chemistry.

### 8. Neutral Baseline

One relaxed panel of the couple standing together.

### 9. Posture Variation

Three panels:

- relaxed together
- shy/reserved together
- confident together

### 10. Wardrobe / Accessory Harmony

Four close-up callouts:

- fabric
- outerwear
- footwear
- accessories

Focus on how the designs work together.

## Reusable Instruction Block

```text
创建一张统一的 COUPLE CHARACTER REFERENCE SHEET。它必须以情侣关系为中心，而不是两张独立角色卡拼在一起。强调浪漫化学反应、共同身份、视觉和谐、身高比例关系和情绪连续性。

版式使用 4:3 横向布局，干净白底或中性背景，清晰英文标签，间距平衡，无水印无 logo。中心 MAIN IDENTITY + SCALE SHEET 必须占主导，展示两人正面、3/4、侧面、背面，并带身高线。右侧展示浪漫表情演进、默契瞬间、头部细节和情侣特写。底部展示中性基线、姿态变化、服装/配饰协调。

所有面板中只能有这两个人，且两人身份、脸、身高、服装、关系气质必须完全一致。整体像高级情侣视觉圣经，技术结构清楚，同时呈现柔和浪漫气质。
```

# Animated Cinematic Character Pitch Board Pattern

Use this pattern when the project needs a high-budget animated cinematic character design sheet for GPT Image 2, suitable for a director, casting team, costume department, animation team, and Seedance 2.0 identity control.

This pattern is distilled from a GPT Image 2 prompt method focused on animated cinematic character designs. Its strongest value is not only "what the character looks like", but how to make the character feel castable, performable, merchandisable, and production-ready.

## Best For

- animated film character pitch boards
- stylized realism characters
- Pixar-like, DreamWorks-like, anime-inspired, semi-realistic, or culturally grounded animated roles
- AI真人剧角色动画化
- recurring hero / villain / sidekick / mascot roles
- characters that need acting range, micro-expression, costume logic, and multi-angle identity consistency
- projects where the character sheet may later guide Seedance, merchandising, pitch decks, or episode continuity

## Core Directive

The sheet must feel art-directed and intentional.

Avoid:

- generic model sheet layout
- evenly spaced grid
- symmetry for symmetry's sake
- auto-generated panel placement
- lifeless neutral poses only
- pretty but uncastable character design
- clothing without story logic
- face drift between angles
- plastic fabric, clean cosplay, or material ambiguity
- text overload that makes the board unreadable

Require:

- slightly asymmetrical premium composition
- clear hierarchy
- actor-like transitional emotion
- micro-expressions
- body language rhythm
- wardrobe and material accuracy
- strict turnaround consistency
- cinematic portrait tied to the character's world
- production notes useful to director, costume, animation, and AI video teams

## Visual Lessons From Reference Boards

Use these layout lessons when adapting the template:

### Human / Live-Action-Stylized Board

For handsome, realistic, thriller, sci-fi, short-drama, or actor-like characters:

- use a premium dark gray studio-board background
- place a large cinematic portrait in the upper center or upper left
- place identity and psychological profile in a narrow left column
- place head study in the upper right
- place full-body turnaround across a wide middle row
- place wardrobe, accessories, and props in the right column
- place material close-ups along the bottom strip
- keep the feeling controlled, elegant, high-end, and production-department ready

This layout is best when face identity and actor casting matter most.

### Stylized Creature / Mascot / Animated Character Board

For creatures, mascots, cartoon-realistic characters, fantasy characters, IP roles, or memorable sidekicks:

- use a more expressive collage layout with taped notes, handwritten-style labels, rough paper panels, and layered production scraps
- make the hero portrait very large and characterful
- add a short motto, quote, or attitude line near the title
- include action / idle poses, not only neutral turnaround
- add "always present" props when a prop is part of the character's identity
- add color and texture guide with material names, not just color swatches
- add a bold production note such as "reference fidelity mandatory" when generated from a user reference

This layout is best when personality, silhouette, charm, and IP memorability matter most.

## Required Modules

### 1. Character Identity

Include:

- full name
- alias / nickname / codename
- age or stylized age
- height
- build, body proportion, and posture tendency
- role function in story
- ethnicity or design language when relevant
- visual style direction
- signature quote, motto, or attitude line if the character is stylized or IP-like

### 2. Face Design

Define:

- face shape
- bone structure
- exaggeration level
- skin tone, texture, imperfections
- eye size, spacing, color, expressiveness
- hair style, texture, movement, imperfections
- unique features such as scars, dimples, moles, teeth shape, brow shape, jawline, freckles, eye bags

### 3. Psychological Profile

Include:

- 3-5 dominant personality traits
- internal conflict: what they want vs what holds them back
- 3 behavior habits that reveal personality
- emotional baseline
- how fast the emotion shifts under pressure

This turns a visual character into a performable role, which helps later dialogue, acting, and Seedance performance prompts.

### 4. Performance Direction

The character must feel like a real actor caught mid-moment, not a mannequin posing.

Define:

- posture tendency
- movement rhythm: stiff, sharp, bouncy, dragging, elegant, nervous, heavy, relaxed
- idle behavior: fidgeting, stillness, tension, scanning the room, touching a prop
- micro-expression anchors: lip tension, eye flicker, brow shift, jaw tightening, controlled breath
- transitional emotion: the expression should feel like it is changing, not frozen
- 2-4 action or idle poses when the character has a strong performance silhouette

### 5. Wardrobe And Props

Wardrobe must reveal character identity and story function.

Include:

- garment 1: fabric type, fit, wear, imperfections
- garment 2: fit, distortion, stitching
- layering logic
- footwear material and wear patterns
- accessories that reveal personality
- props that reinforce who they are
- why each visible item belongs to this character
- always-present props that must never disappear during video generation

### 6. Material Accuracy

Show:

- fabric stretch
- stitching
- wrinkles
- wear marks
- aging
- dirt, smudges, scratches, sweat marks, or usage traces
- soft skin-light interaction
- specific surface behavior for leather, denim, cotton, metal, plastic, glass, rubber, armor, fur, hair, jewelry

No plastic look unless the character intentionally has plastic material.

Add a color and texture guide when the character has distinctive materials. This should include labeled swatches such as skin, denim, clothing black, metal, leather, smoke, fur, armor, neon, or dust.

### 7. Strict Turnaround

Include:

- full-body front view
- full-body 3/4 front view
- full-body side view
- full-body back view
- full-body 3/4 back view

Every view must keep identical:

- face identity
- height and body proportion
- hairstyle
- costume structure
- shoes
- accessories
- material wear marks
- asymmetric details

### 8. Head Study

Include:

- front neutral
- 3/4 primary personality
- profile structure
- looking down
- looking up
- dynamic intense angle

All expressions should feel mid-thought, not posed.

### 9. Cinematic Portrait

Include one larger portrait panel:

- environment tied to character
- practical light source
- contrast level
- warm / cool / stylized color tone
- specific narrative moment
- 50mm or 85mm lens feeling
- shallow depth of field
- cinematic realism or stylized realism

### 10. Layout And Production Notes

The board should use:

- clean art-directed sheet layout
- neutral gray or warm gray background
- height scale
- annotation callouts
- wardrobe breakdown
- production notes
- color swatches
- color and texture guide
- optional handwritten note, taped note, or signature quote for stylized boards
- premium studio pitch-board feeling

Keep labels readable. Use short bilingual labels when useful.

### 11. Production Fidelity Note

If the character is based on a user-uploaded reference image, add a visible note:

```text
REFERENCE IMAGE FIDELITY MANDATORY.
Do not deviate from the uploaded reference.
All angles, expressions, materials, props, and details remain 100% consistent.
```

If there is no uploaded reference, replace this with:

```text
CONTINUITY FIDELITY MANDATORY.
All angles, expressions, materials, props, and details remain 100% consistent.
```

## Reusable Image2 Prompt Block

```text
创建一张电影制作级动画角色设计提案板，用于导演、选角团队、服装部门、动画团队和 AI 视频制作团队。角色名称：[YOUR CHARACTER NAME HERE]。画面必须像高预算动画电影 pitch board，不是普通模型表、AI 写真或海报。

CORE DIRECTIVE / 核心指令：
拒绝普通等距网格，拒绝机械对称，拒绝为对称而对称。构图必须有艺术指导感、主次层级和轻微不对称，所有区域都像被专业美术指导精心安排过，而不是自动生成。角色必须像真实演员在叙事瞬间被捕捉到，而不是摆拍模型。

CHARACTER IDENTITY / 角色身份：
Name：[全名] | Alias：[昵称/代号] | Age：[真实或风格化年龄] | Height：[身高] | Build：[体型、比例、姿态倾向] | Role：[故事功能] | Ethnicity / Design Language：[文化背景或设计语言] | Style：[Pixar-style stylized realism / semi-realistic cinematic animation / anime-inspired / culturally grounded 等] | Signature Quote：[一句能锁住角色态度的短台词或座右铭]

FACE DESIGN / 面部设计：
Structure：[脸型、骨骼结构、夸张程度] | Skin：[肤色、质感、瑕疵] | Eyes：[大小、间距、颜色、表达力] | Hair：[发型、质地、运动感、瑕疵] | Distinct Features：[疤痕、酒窝、痣、雀斑、牙齿形状、眉形、眼袋、下颌线等独特锚点]

PSYCHOLOGICAL PROFILE / 心理档案：
Core Traits：[3-5 个主导性格] | Internal Conflict：[想要什么 vs 被什么阻碍] | Behavior Patterns：[3 个暴露性格的习惯] | Emotional Baseline：[默认情绪 + 情绪变化速度]

PERFORMANCE DIRECTION / 表演指导：
角色必须有演员感和过渡情绪。需要微表情：嘴唇张力、眼神闪动、眉毛变化、下颌紧张、克制呼吸。避免僵硬摆拍和刻意对称。
Body Language：[姿态倾向] | Movement Rhythm：[僵硬/锐利/弹跳/拖沓/优雅/紧张/沉重/松弛] | Idle Behavior：[小动作、静止、紧绷、扫视环境、触摸道具等]。如果角色是风格化/IP/生物角色，加入 ACTION / IDLE POSES：2-4 个动作或待机姿势，表现角色的身体节奏、招牌姿态和可动画化魅力。

WARDROBE / 服装道具：
Garment 1：[面料、版型、磨损、瑕疵] | Garment 2：[合身度、变形、缝线] | Layering Logic：[叠穿逻辑] | Footwear：[材质、磨损痕迹] | Accessories：[体现性格的配饰] | Props：[强化身份的道具] | Always Present Props：[永远不能消失的身份道具]。每件物品都必须有角色逻辑，不要随机漂亮衣服。

MATERIAL ACCURACY / 材质准确性：
面料必须显示弹性、缝线、褶皱、磨损和使用痕迹。皮肤必须有柔和光线反应。加入污渍、划痕、老化、汗渍、灰尘或使用痕迹。除非是角色设定需要，否则不要塑料质感。
加入 COLOR & TEXTURE GUIDE / 色彩与材质指南：用清晰色块和材质块标注皮肤/毛发/布料/牛仔/皮革/金属/烟雾/VFX/灰尘等关键材质。色块不只是颜色，要写出材质含义。

TURNAROUND / 严格转面：
全身正面、3/4 正面、侧面、背面、3/4 背面。所有角度必须保持同一张脸、同一体型比例、同一发型、同一服装结构、同一鞋履、同一配饰、同一材质磨损和同一不对称细节。任何角度都不得脸漂移或服装漂移。

HEAD STUDY / 头部研究：
Front neutral / 正面中性，3/4 primary personality / 3/4 主要性格，Profile / 侧面结构，Looking Down / 向下看，Looking Up / 向上看，Dynamic Intense Angle / 强烈动态角度。所有表情都是思考中的瞬间，不是摆拍。

CINEMATIC PORTRAIT / 电影肖像：
加入一张较大的电影肖像，环境：[与角色绑定的地点]，灯光：[实际光源、对比度]，色调：[暖/冷/风格化]，表情：[具体叙事瞬间]，镜头：[50mm 或 85mm、浅景深、电影写实或风格化写实]。

LAYOUT / 版式：
根据角色类型选择版式：
如果是写实/真人/短剧/科幻惊悚角色，使用高端深灰制片厂技术板：左侧身份和心理档案，中上大幅电影肖像，右上头部研究，中部大范围转面，右侧服装/配饰/道具，底部材质细节条。
如果是风格化/IP/生物/吉祥物角色，使用更有性格的拼贴式角色圣经：大幅英雄肖像、贴纸便签、手写感短标签、招牌台词、转面、头部研究、动作/待机姿势、电影肖像、永远存在的道具、色彩与材质指南、生产备注。
无论哪种版式，都必须干净、有艺术指导感、文字短而可读，不要大段小字。整体适合电影开发、动画制作、AI 视频连续性、周边设计和 pitch deck。

CONSISTENCY RULE / 一致性规则：
面部、比例、服装、材质、配饰和细节在所有视图中必须完全一致。不得在不同角度重新设计。不得出现多张不同脸、随机换装、比例漂移、手部畸形、额外肢体、模糊细节、水印或乱码文字。
在画面显眼位置加入 PRODUCTION NOTES / 生产备注：REFERENCE IMAGE FIDELITY MANDATORY（如果使用用户上传图）或 CONTINUITY FIDELITY MANDATORY（如果无上传图）；All angles, all expressions, all materials, all props, all details 100% consistent。

OUTPUT / 输出质量：
极高细节，清晰焦点，生产级保真度，高情感可读性，柔和几何造型，电影级光影，premium animated film pitch board。
```

## When To Use With Other Branches

Use this instead of `master-character-reference-sheet.md` when the character needs stronger art direction, acting psychology, stylized animated film appeal, or pitch-board quality.

Use `master-character-reference-sheet.md` when the top priority is a clean Seedance-readable technical identity sheet.

Use `cinematic-character-card-generator.md` for AI真人剧 or photo-based transformations where actor identity preservation and costume realism matter more than animated-film appeal.

Use `ue5-metahuman-technical-sheet.md` for highly technical hard-surface, armor, asymmetry, body landmark, hands/feet, or game-character continuity.

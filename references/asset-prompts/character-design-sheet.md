# Character Design Sheet Pattern

Use this pattern when the user needs stable recurring characters for Seedance 2.0, especially AI live-action drama, webtoon episodes, monsters, creatures, IP roles, or any video where identity drift would be costly.

## Applicability

Use for:

- main character
- recurring supporting character
- villain
- creature/monster
- mascot/IP character
- actor-like live-action role

Do not use for:

- one-frame extras
- background crowds
- characters whose identity does not need continuity

## Core Pattern

The Image2 board should feel like a film-production character design sheet for directors, casting, costume, performance, and AI video continuity. It should not look like a generic uniform model sheet.

Must include:

- character code, name/role, age range, height, body type, ethnicity/design language when relevant
- asymmetric art-directed layout, no generic equal grid, no bland symmetry
- turnarounds: front, 3/4 front, pure side, back, dynamic angle
- face design: structure, skin texture, eyes, hair, scars, distinctive features
- head study: neutral, 3/4, side, looking down, looking up, dynamic emotional angle
- expression range: baseline, anger, fear, pain, restraint, micro-expression
- psychological profile: core trait, internal conflict, behavioral pattern, emotional baseline
- performance direction: actor-like realism, body language, idle behavior, gesture habits
- costume and texture lock: outfit, shoes, accessories, hero prop, fabric/material precision, stains/weathering
- cinematic portrait: environment-lit portrait with camera/lens/lighting notes
- wardrobe/prop breakdown: isolated clothing pieces and key personal items with labels
- color palette and forbidden drift labels

## Live-Action Human Additions

Prioritize actor continuity:

- realistic skin pores
- uneven skin tone
- natural hairline
- eye fatigue
- clear face from every angle
- no AI doll face
- performance baseline for emotional continuity

## Creature/Monster Additions

Adapt the same board logic:

- anatomy identity
- head shape
- silhouette
- scale
- skin/fur/scale material
- wounds/scars
- movement posture
- behavioral baseline: predatory, protective, wounded, intelligent, berserk, etc.
- cinematic portrait and dynamic angle remain mandatory

## Reusable Instruction Block

```text
这必须像高预算电影项目的导演级角色企划板，供导演、选角/动画/VFX/服装/表演部门和 Seedance 2.0 参考使用。不要通用模型表，不要均匀网格，不要对称排版，不要海报。版式要有艺术指导感：左侧大体块展示转身，中部展示头部研究与表演，右侧展示电影肖像和动态姿态，底部展示材质、比例、色卡、心理/行为档案和禁止漂移项。所有标签必须大号清晰，Seedance 能读懂。
```

## Drift Labels

Use short, large labels:

- DO NOT CHANGE FACE
- DO NOT CHANGE BODY PROPORTION
- KEEP HAIRLINE
- KEEP COSTUME
- KEEP SCAR / KEEP KEY FEATURE
- PERFORMANCE BASELINE
- NO AI DOLL FACE

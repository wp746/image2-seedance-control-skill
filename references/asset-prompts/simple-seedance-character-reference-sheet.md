# Simple Seedance Character Reference Sheet Pattern

Use this pattern when Seedance mainly needs a clean identity reference for a stylized 3D character, robot, mascot, creature, or simple recurring role.

This is lighter than a director-level character design sheet. It prioritizes clarity and consistency over decoration.

## Best For

- stylized 3D animated short characters
- robots
- mascots
- simple creatures
- single-character shorts
- Seedance identity reference image
- characters whose design must stay readable at small scale

## Core Intent

Create a clean production character sheet used as a Seedance 2.0 character reference.

Clarity and consistency matter more than decoration.

## Layout

Use:

- horizontal 16:9 character sheet
- clean neutral background
- thin panel borders
- large readable labels
- no dense text
- no tiny annotations
- no logos
- no watermark

## Required Sections

### Section 1: Turnaround

Five full-body views of the exact same design:

1. FRONT
2. 3/4 FRONT
3. SIDE
4. BACK
5. 3/4 BACK

Keep proportions identical across all views. Use neutral standing pose, relaxed arms, planted feet.

### Section 2: Detail Close-Ups

Use four to six small clean boxes for the most important identity anchors:

- eye/camera/face feature
- chest/core/emblem/light
- joint/hand/foot structure
- color marking/scar/symbol
- key texture/material
- hair/groom/antenna/tail if relevant

### Section 3: Action Poses

Use three simple poses that cover story use:

- holding or using key prop
- movement pose
- emotional/resting/failure pose

### Bottom: Color Swatches

Small swatches with short labels only.

## Reusable Image2 Instruction Block

```text
创建一张干净的生产角色参考表，用作 Seedance 2.0 角色身份参考。清晰度和一致性比装饰性更重要。16:9 横向角色表，干净中性背景，简单细边框，大号可读标签，不要密集文字，不要小字注释，不要 logo，不要水印。

版式分三部分：
SECTION 1: TURNAROUND，包括 FRONT、3/4 FRONT、SIDE、BACK、3/4 BACK，必须是完全同一个角色设计，身体比例一致，中性站姿，手臂放松，双脚站稳。
SECTION 2: DETAIL CLOSE-UPS，展示最重要的身份锚点和材质细节。
SECTION 3: ACTION POSES，展示三个故事中会用到的动作姿态。
底部放短标签色卡。

不要环境场景，除非需要淡淡的中性地面阴影。不要额外角色。不要无关道具。整张图必须简洁、清楚、适合作为 AI 视频生成的角色身份参考。
```

## Continuity Labels

Use short labels:

- SAME DESIGN ALL VIEWS
- KEEP BODY PROPORTIONS
- KEEP COLOR MARKINGS
- KEEP MATERIAL WEAR
- NO EXTRA PROPS
- NO LOGOS
- NO IDENTITY DRIFT

## When To Prefer This Pattern

Prefer this over the larger character-card patterns when:

- the character is simple and stylized
- the project needs fast production
- the reference must be very readable for Seedance
- the character is a robot, mascot, or small creature
- expressive psychology and wardrobe breakdown are less important

Use `character-design-sheet.md` or `cinematic-character-card-generator.md` for human drama roles, complex costumes, or actor-like performance.

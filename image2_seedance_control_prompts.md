# Image2 Seedance Control Prompts

## Project Settings
- 片名: 雨夜霓虹 / Rain Neon
- 类型: 高端香水广告
- 画幅: 9:16 竖屏
- 总时长: 15 秒
- 风格: 暗调电影质感 — 深黑基调 / 霓虹色彩点缀 / 雨中镜面反射 / 王家卫式孤独氛围 / 35mm胶片颗粒感
- 生产策略: 单段 15 秒，1 组资产板 + 1 块故事板 + 1 段 Seedance prompt 对

## Director Recommendation

这个创意的核心情绪是"城市孤独中的私密时刻"——不是卖香水，是卖那个在雨中停下来闻自己手腕的瞬间。

节奏设计：
- 不是在赶路，是在漂移。镜头像另一个行人，保持距离，不打扰。
- 前 6 秒建立世界（雨、霓虹、人影），中 5 秒产品露出和私密动作（喷香水、闻手腕），后 4 秒情绪收束（她继续走，但变了）。
- **只用 4 个镜头。** 不是 12 个。要的是沉浸，不是快速切换。

## Usage

如果你已经有角色、场景、道具资产图，先把用户原始资产图与下方的"资产设计提示词"一起放进 GPT Image 2，生成正规的资产连续性板。再复制"故事板提示词"去 GPT Image 2 出镜头时间轴控制图。进 Seedance 2.0 时，把用户原始资产图、正规资产板、故事板控制图一起作为全能参考上传，然后复制该故事板后面对应的"Seedance 2.0 视频提示词"。

**注意**：不要只输入"参考故事板生成"。故事板上只有逻辑标签和视觉标注，文字极少。必须使用逐镜头、逐秒的 Seedance 文字提示词补全板上画不出的所有细节，形成互补双控。

## Production Control / 工业化连续性控制
- 项目圣经: 15 秒单段，无需圣经
- 角色状态: CHAR_A — 湿发、黑风衣、透明伞、左手撑伞、右手持香水瓶
- 场景空间: SCENE_01 — 涩谷或新宿窄巷，霓虹灯密集，路面反光，积水
- 道具状态: PROP_PERFUME — 方形玻璃瓶、金属盖、深琥珀色液体、瓶身带雨滴
- 风格/光线: 深黑基调 + 霓虹粉紫蓝青色点缀 + 雨面镜面反射 + 35mm 胶片颗粒
- 声音/VFX: 雨声、脚步在水面、霓虹电流嗡鸣、香水瓶打开时的细微机械声、钢琴单音
- 情绪递进: 疏离 → 停顿 → 私密 → 释然
- 镜头接缝: 每镜 IN/OUT 状态见下方故事板 + Seedance prompt
- 禁止漂移: 脸、发型（湿发状态）、风衣、伞、香水瓶、霓虹颜色配比、时间/天气

```text
脸型：鹅蛋偏窄，下颌线清晰但不锋利。颧骨略高，亚洲人骨骼结构。

版式要求：
- 非对称艺术指导排版，左侧大体块展示转身，中部头部研究与微表情，右侧电影肖像+动态姿态，底部材质/比例/色卡/心理档案/禁止漂移
- 所有标签大号清晰，Seedance 能读懂

必须包含：
转身图：全身正面 → 3/4 前侧 → 纯侧 → 背面 → 动态角度（撑伞转身回眸），所有视图同一人物同一服装

头部研究：中性正面 → 3/4 侧 → 纯侧 → 低头 → 抬头 → 回眸情绪角度（眼含微光）
微表情范围：基准面（疏离但非冷漠）→ 停顿（眼神定住，嘴唇微启）→ 私密（闻香水时眉弓微降、眼睑半闭、嘴角几乎不察觉的上扬）→ 释然（表情比基准面柔软了一个灰度）
心理档案：核心特质——内敛但非压抑。内在冲突——城市疏离 vs 渴望被感知。行为模式——独处时放下面具，公共空间即恢复防御。
表演指导：不是表演，是存在。不要"演"悲伤或"演"释然。微表情幅度控制在 2mm 级别。眨眼频率低于常人（疏离感），闻香水时闭眼时长 1.2 秒（私密时长）。
服装锁定：黑色中长款风衣，哑光防水面料，腰部系带，立领。内搭深灰高领针织。黑色直筒裤，皮质短靴（跟高 4cm，不是细高跟）。透明长柄雨伞，塑料质地，雨滴附着在伞面。左手撑伞，右手自由。

道具关系：右手持香水瓶（PROP_PERFUME），持握方式——拇指在瓶身侧面，食指和中指轻触瓶盖，其余手指自然弯曲。手腕内侧喷洒位置可见微湿光泽。
颜色板：黑色（风衣/裤子）→ 深灰（内搭）→ 琥珀深棕（香水液体）→ 银色（瓶盖）→ 透明（雨伞）
禁止漂移标签（大号）：DO NOT CHANGE FACE / 保持发型（湿发状态）/ 保持服装 / 保持气质（疏离非冷漠）/ 保持身高比例 / PERFORMANCE BASELINE（2mm微表情）/ NO AI DOLL FACE
```

### Prompt A02 — Scene Asset Board — SCENE_01

```text
这是给 Seedance 2.0 读取的场景资产连续性板，用于锁定空间物理现实。不是风景海报，不是概念图。

SCENE_01 / 东京雨夜窄巷
位置：像涩谷或新宿后巷——两侧 4-6 层建筑夹出的窄巷，宽度约 4 米。建筑外立面有空调外机、管道、消防梯、日文招牌。

必须包含：
场景编号：SCENE_01
同一空间 4 个角度：
- 角度 1：从巷口向深处看——两侧建筑夹出纵深，地面反光积水，远处巷口有微弱街灯光
- 角度 2：从巷子中部向巷口回看——可见来路和巷口霓虹光晕
- 角度 3：仰角——天空被建筑切成长条，雨丝从上方斜落，建筑轮廓上有霓虹灯管
- 角度 4：近地面——积水反光中的霓虹倒影，路面沥青纹理，水波纹
所有角度展示同一空间、同一建筑结构、同一光线逻辑

入口和出口：
- 入口：巷口，画面后方偏左，远处街灯和车灯光晕
- 出口：巷子深处，画面前方偏右，越来越暗

摄影机安全区：
- CAM 1：巷子右侧，距墙面 1.5m，中景机位
- CAM 2：巷子中央偏左，手持跟随，距人物 3m
- CAM 3：巷子右侧靠墙，近景机位，距人物 1.5m
- CAM 4：巷子中央偏后，低角度仰拍，距人物 5m

主光源和氛围：
- 主光源 1：右侧建筑二楼的粉紫色霓虹灯管（持续亮，不闪烁）——硬光，边缘清晰，投射在人物右侧颧骨和肩部

- 环境光：远处巷口街灯（暖黄钠灯，2700K），通过雨雾散射，作为背景深度光

关键固定物位置：
- 右侧建筑二楼空调外机：距地面 3.5m，CAM 1 上方
- 左侧建筑消防梯：距巷口 8m，金属锈蚀，雨滴沿扶手滴落
- 地面井盖：巷子中部偏左，积水在边缘形成小漩涡
- 日文招牌：右侧建筑一楼，白色亚克力灯箱，日文，持续亮

颜色板：深黑（建筑阴影）→ 湿沥青灰 → 粉紫霓虹（右侧主光）→ 青色霓虹（左侧补光）→ 暖黄街灯（远处背景深度）→ 银白雨丝

大号禁止漂移标签：DO NOT FLIP LAYOUT / KEEP DOOR & WINDOW POSITIONS / DO NOT CHANGE WEATHER / KEEP LIGHT DIRECTION / DO NOT MOVE FURNITURE & FIXTURES / DO NOT CHANGE TIME OF DAY (凌晨 1:00)
```
